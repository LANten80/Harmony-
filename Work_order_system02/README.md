# 工单管理应用（HarmonyOS版）

基于 HarmonyOS 4.0+ 和 ArkTS 开发的工单管理系统应用。

## 项目概述

这是一个完整的工单管理应用，实现了工单的创建、查看、编辑、进度追踪和统计等功能。采用声明式开发范式，使用 ArkTS 语言和 ArkUI 框架构建。

## 项目结构

```
entry/src/main/ets/
├── pages/                    # 页面目录
│   └── MainPage.ets         # 主页面
├── view/                     # 视图组件目录
│   ├── TargetInformation.ets    # 整体进度展示组件
│   ├── TargetList.ets           # 工单列表组件
│   ├── TargetListItem.ets       # 工单项组件
│   ├── ProgressEditPanel.ets    # 进度编辑面板
│   └── AddTargetDialog.ets      # 添加工单弹窗
├── viewmodel/                # 数据模型目录
│   ├── TaskItemModel.ets     # 工单实体模型
│   └── DataModel.ets         # 数据操作模型
└── common/                   # 公共资源目录
    ├── constants/            # 常量定义
    │   └── CommonConstant.ets
    └── utils/                # 工具类
        ├── DateUtil.ets      # 日期工具
        └── Logger.ets        # 日志工具
```

## 核心功能

### 1. 工单管理
- ✅ 工单创建：支持通过弹窗添加工单
- ✅ 工单查看：列表视图展示所有工单
- ✅ 工单编辑：支持更新工单进度
- ✅ 工单删除：支持删除工单（扩展功能）

### 2. 进度管理
- ✅ 进度设置：通过滑块和按钮调整进度（0-100%）
- ✅ 进度追踪：实时显示进度变化
- ✅ 进度可视化：环形进度条展示整体进度

### 3. 统计与分析
- ✅ 总体统计：总工单数、完成数、进行中数
- ✅ 完成率计算：自动计算完成百分比
- ✅ 实时更新：数据变化时自动更新统计

### 4. 用户交互
- ✅ 展开/折叠：点击工单项展开进度编辑面板
- ✅ 互斥展开：同一时间只允许一个工单项展开
- ✅ 编辑模式：支持编辑模式切换和批量选择
- ✅ 动画效果：流畅的展开/折叠动画

## 技术特性

### 状态管理
- 使用 `@State` 管理组件内部状态
- 使用 `@Prop` 实现父向子数据传递
- 使用 `@Link` 实现父子组件双向同步
- 使用 `@Provide/@Consume` 实现跨组件状态共享
- 使用 `@Watch` 监听状态变化

### 数据模型
- 单例模式管理数据，确保数据一致性
- 支持数据变更监听和自动更新
- 提供完整的 CRUD 操作接口
- 自动计算统计数据

### 组件化设计
- 高度模块化的组件设计
- 清晰的组件职责划分
- 可复用的组件接口
- 良好的组件通信机制

## 开发环境要求

- DevEco Studio 6.0 或更高版本
- HarmonyOS SDK API 12 或更高版本
- Node.js 环境（用于包管理）

## 运行说明

1. 使用 DevEco Studio 打开项目
2. 连接 HarmonyOS 设备或启动模拟器
3. 点击运行按钮，应用将自动安装并启动

## 主要文件说明

### MainPage.ets
主页面组件，作为应用的入口页面，集成所有功能组件，管理全局状态。

### TaskItemModel.ets
工单实体模型，定义工单的基本属性和业务逻辑。

### DataModel.ets
数据操作模型，采用单例模式，提供数据的增删改查和统计功能。

### TargetInformation.ets
整体进度展示组件，显示目标信息和整体进度统计。

### TargetList.ets
工单列表组件，展示工单列表，支持编辑模式和添加操作。

### TargetListItem.ets
工单项组件，显示单个工单信息，支持展开/折叠和进度编辑。

### ProgressEditPanel.ets
进度编辑面板，提供进度设置和操作按钮。

### AddTargetDialog.ets
添加工单弹窗，提供工单名称输入和确认操作。
